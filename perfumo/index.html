Technical Specifications

Architecture

  • Framework: React Native + Expo SDK 53
  • Database: SQLite (embedded, 24K+ records)
  • State Management: Zustand + React Context
  • Styling: NativeWind (Tailwind CSS)

Integrations

  • AI: OpenAI GPT-4 API
  • Weather: WeatherAPI.com
  • Authentication: Google & Apple OAuth
  • Payments: RevenueCat + In-App Purchase

Compatibility

  • iOS: 13.0+ (iPhone & iPad)
  • Android: API Level 21+ (Android 5.0+)
  • Storage: ~50MB app + database
  • Offline: Full functionality without internet

Privacy & Security

  • Data Storage: Local-first (AsyncStorage + SQLite)
  • Cloud Sync: Optional, user-controlled
  • Encryption: All network traffic encrypted (HTTPS)
  • Privacy: No tracking, minimal data collection
